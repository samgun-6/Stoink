What is Stoink

Stoink for Stock Price Prediction is a system that uses deep learning LSTM model to predict the price increase or decrease of one or more stocks for the next three months.

What is LSTM

The deep learning model LSTMï¼ˆLong Short Term Memory) is based on RNN (Recurrent Neural Network) structure, which can remember each information throughout time series, however it has weak impact on long-term memory compared to short-term memory. RNN also exists both exploding and vanishing gradient issue that cause the model to be unstable. LSTM solves the issue that the RNN model has to do with long memory of time series. Thus, it can well describe long memories of time series in predicting stock prices.
